# Coffee-Shop-Sales-Analysis

This repository contains a detailed sales analysis for a coffee chain with 3 location from January to July 2023.
The goal is to understand revenue trends by time and by store location.

Summary of Finding
Revenue constantly increased from February to June.
* Jan-12%
* Feb-11%
* Mar-14%
* Apr-17%
* May-22%
* Jun-24%

# Revenue by Location
Three stores performed slightly different

* Night
